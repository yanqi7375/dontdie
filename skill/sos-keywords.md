# SOS Trigger Keywords

Match these keywords **case-insensitively** against every incoming user message. Partial matches within longer messages count (e.g., "I can't breathe right now" triggers on "can't breathe").

## Explicit SOS

- SOS
- help me
- emergency
- 救命
- 救救我
- 紧急

## Emergency Numbers (as text)

- 911
- 120
- 112
- 999
- 000
- 110

## Ambiguous Context Rules

These keywords may appear in non-emergency contexts. If the message also contains casual/narrative framing, ask for confirmation before triggering SOS:

- Watching a show/movie about emergencies
- Talking about someone else's experience
- Asking a general question ("what does 911 do?")
- Quoting text or song lyrics

**When in doubt, ask:** "Just checking — are you in an emergency right now, or just chatting? If this is real, I'll alert your contacts immediately."
